----------------------------------------------------------------
HARDWARE NOTE
----------------------------------------------------------------
 
The models (Alpha, Beta, Gamma) run via HuggingFace Transformers and
will automatically use whatever hardware is available:
 
  CUDA GPU   →  fast, recommended (~8 GB VRAM minimum)
  Apple MPS  →  works on Apple Silicon (M1/M2/M3)
  CPU only   →  works but slow — expect longer inference times
 
To explicitly target a specific GPU (if you have multiple):
  CUDA_VISIBLE_DEVICES=0 python <script>.py   # GPU 0
  CUDA_VISIBLE_DEVICES=1 python <script>.py   # GPU 1
 
To force CPU only:
  CUDA_VISIBLE_DEVICES="" python <script>.py
